## For builders

1. Pick an idea from the [board](https://ideas.ummah.build).
2. Use the [Idea dissection](templates/idea-dissection.md) template to define problem, solution, and scope.
3. If it touches worship, finance, or content: run the [Islamic authenticity](templates/islamic-authenticity.md) checklist.
4. Follow [Workflow 1 (Idea → MVP)](workflows/README.md#workflow-1-idea--mvp-solo-or-small-team) or the [Quick start](workflows/quick-start.md) for a minimal path.
5. Before launch: use the [Launch checklist](templates/launch-checklist.md).
